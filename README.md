# Gua
Welcome to Gua programming language based on [LLVM](https://llvm.org) and [antlr](https://github.com/antlr/antlr4/) compiled in cpp

> For now, Gua is not released (under development).

> [!IMPORTANT]
> Gua is under a LISCENSE [GNU General Public License](https://github.com/KeyeFauV2/Gua/blob/main/LICENSE).

## Why Gua ?
* GUA is intended to modernize and simplify CPP.
* I take [lua](https://www.lua.org) and [C ++](https://en.wikipedia.org/wiki/C%2B%2B) and mix them to make gua.

## Example of GUA :
```lua
var a = "b"
Console.print(a)

local function subtract(a, b)

return a + b
end
```
